# Sealegs

A 2D pirate-themed platformer where players traverse a segmented ship while
experiencing challenges inspired by vestibular disorders. Across three levels,
players navigate the ship, balancing risk, perception, and movement under
unstable sensory conditions.

> Current build: core player movement and the overall screen/level structure
> only. Hazards, platforms, and treasure are not yet implemented.

## Setup and Interaction Instructions

hi
To run the sketch locally, open `index.html` in Google Chrome using Live Server.

**Controls:**

- `WASD` — move
- `ENTER` — start the game / return to title
- `N` — advance to the next level (placeholder until goal logic exists)
- `L` — trigger the lose screen (placeholder until hazards exist)
- `R` — retry the current level from the lose screen

**Opening the Chrome Console**

- **Windows:** Press `F12` or `Ctrl + Shift + J`, then click the **Console** tab
- **Mac:** Press `Cmd + Option + J`

The console will show any errors in your sketch.
